# RAG Docs Assistant

This project is an end-to-end Retrieval-Augmented Generation (RAG) system built from scratch.

## Goals

- Learn RAG fundamentals without frameworks
- Build full pipeline: ingestion → retrieval → generation
- Use documentation as knowledge source
- Use local embedding models

## Scope (MVP)

- Ingest documentation pages
- Clean and chunk content
- Generate embeddings locally
- Store vectors
- Retrieve relevant chunks
- Answer user questions with sources
